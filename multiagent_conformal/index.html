The Full Pipeline
01
Environment + Noise

Each agent gets a noisy observation of the other's true position. Noise is Gaussian with σ=1.5 cells. This is the source of all uncertainty in the system.

02
Classifier (Logistic Regression)

A model trained on 20,000 (observation → true position) pairs. Takes a noisy (obs_x, obs_y) and outputs a softmax probability distribution over all 100 grid cells.

03
Calibration (Conformity Scores)

On 4,000 held-out examples, record the probability given to the correct true label. This fingerprints the model's real-world accuracy before any test-time predictions.

04
Threshold q (Quantile)

From the calibration scores, compute q = α-th percentile. This is the cutoff: include any cell with probability ≥ q in the prediction set.

05
Prediction Set (Coverage Guarantee)

At test time: C(obs) = all cells i where P(i | obs) ≥ q. The guarantee: P(true position ∈ C) ≥ 1−α. Proven under exchangeability.

06
Minimax Policy

For each possible action, compute the worst-case cost assuming the other agent is at the worst position in the prediction set. Pick the action that minimizes worst-case cost.

Key Formulas

Conformity Score

s_i = f(obs_i)[y_i]

The probability the classifier assigned to the true label on calibration example i.

Threshold (finite sample correction)

q = Quantile(scores, ⌈(n+1)·α⌉ / n)

The (n+1) correction makes coverage exact rather than approximate for finite calibration sets.

Prediction Set

C(obs) = { i : f(obs)[i] ≥ q }

All classes whose probability meets or exceeds the threshold. Guaranteed to contain the true label with probability ≥ 1−α.

Minimax Policy

a* = argmin_a [ max_{p ∈ C} cost(a, p) ]

Choose the action whose worst-case outcome (over all positions in the uncertainty set) is least bad. Conservative but provably safe.

Coverage Guarantee

P(y_test ∈ C(obs_test)) ≥ 1 − α

Proven under exchangeability of calibration and test data. Not a heuristic — a mathematical theorem.

The α Tradeoff

α controls the coverage-vs-set-size tradeoff. Lower α demands higher coverage, pushing q lower, making prediction sets larger. Higher α allows smaller sets but accepts more misses. In safety-critical settings, α is determined by domain requirements — not math. A robot near humans needs α = 0.01. A recommendation system might tolerate α = 0.30.

Architecture — 5 Layers

Layer 1 — environment.py

10×10 grid, two agents, Gaussian noise model (σ=1.5). Defines the observation function obs = true_pos + ε.

def observe(true_pos, rng, sigma=1.5): noise_x = rng.normal(0, sigma) noise_y = rng.normal(0, sigma) return clip_to_grid((round(x + noise_x), round(y + noise_y)))

Layer 2 — classifier.py

Generates 20,000 (observation, true position) training pairs. Trains logistic regression with softmax over 100 classes. Splits into train / calibration / test sets.

# 100 classes (one per grid cell), 2 features (obs_x, obs_y) clf = LogisticRegression(solver='lbfgs', max_iter=1000) clf.fit(X_train, y_train) probs = clf.predict_proba(X_calib) # shape (n, 100)

Layer 3 — conformal.py

Computes conformity scores on the calibration set, derives threshold q using the finite-sample corrected quantile, produces prediction sets at test time with provable coverage.

scores = probs[np.arange(n), y_calib] # true label probs q = np.quantile(scores, ceil((n+1)*alpha)/n) pred_set = [i for i, p in enumerate(probs) if p >= q]

Layer 4 — policy.py

Minimax policy: for each action, compute max cost over all positions in the prediction set. Pick the action minimizing worst-case cost. Collision penalty = 1000, goal cost = Manhattan distance.

best = min(actions, key=lambda a: max(cost(next_pos(a), p, goal) for p in pred_set))
